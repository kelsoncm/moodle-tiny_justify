# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this plugin, please report it by opening a [GitHub Security Advisory](https://github.com/cte-zl-ifrn/moodle-tiny_justify/security/advisories/new) rather than a public issue.

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Any suggested fix (if available)

You can expect an acknowledgement within 5 business days and a resolution timeline once the vulnerability has been confirmed. If the vulnerability is accepted, a patched release will be made available as soon as possible. If it is declined, you will receive an explanation.
